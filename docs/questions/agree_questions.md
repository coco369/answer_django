
### 点赞面试题接口

***

#### 请求地址: /api/back/questions/[id]/agree/

#### 请求方式：POST

#### 请求参数

    id 面试题id int 必填
    
*** 

#### 响应结果

##### 成功响应

    {
        "code": 200,
        "msg": "请求成功",
        "data": {
            "is_like": 3
        }
    }

##### 失败响应

面试题已点赞的情况

    {
        "code": 2003,
        "msg": "喵了个喵，您已对该面试题点过赞啦，不用再点人家啦",
        "data": {}
    }
